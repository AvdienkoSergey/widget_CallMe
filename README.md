## Widget install

1. Copy repositories: 
  > git clone https://github.com/AvdienkoSergey/widget_CallMe.git
2. Next
  > npm install
  
  > npm run dev:client

  > npm run dev:server

3. Open your browser:
  > http://localhost:3000/demo
